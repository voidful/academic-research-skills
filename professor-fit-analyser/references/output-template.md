# Output Template

Use this as the default structure. Adapt depth to corpus size, but preserve the section order.

**Language**: Output in Traditional Chinese (繁體中文) by default. Keep paper titles, venue names, method names, and technical terms in English. Switch to English if the user writes in English.

**Applicant Level**: Identify the applicant's level (PhD / master's thesis / master's coursework / undergraduate capstone-thesis-專題 / RA / postdoc / collaborator / other) and calibrate all advice accordingly.

# Paper-by-Paper Analysis

- resources (field-appropriate resource types)
- authorship analysis
- impact and signal
- writing quality (adapted to the field's scholarly conventions)

# Fit Analysis for the Applicant
- applicant level (PhD / master's / undergraduate / RA / postdoc / other)
- research role mapping (include "interpretive scholar" for humanities, "clinical/translational researcher" for biomedical, "qualitative researcher" for social sciences, "apprentice learner" for undergraduates)

- professor-student type matching

- field-specific fit considerations (e.g., theoretical alignment for humanities, lab resources for biomedical, clinical access for clinical research)

- probability of traction if contacting now
- best positioning strategy
